## 기능 구현 사항

- 보유 차량
  - Sonata 
    - 대수: 2대
    - 연비: 10km/리터 
  - Avante 
    - 대수: 1대
    - 연비: 15km/리터
  - K5 
    - 대수: 2대
    - 연비: 13km/리터
- 고객이 인터넷으로부터 예약할 때 여행할 목적지의 대략적인 이동거리를 입력 받을 수 있다.
- 이동거리를 활용해 차량 별로 필요한 연료를 주입할 수 있다.
- 차량 별로 주입해야 할 연료량을 확인할 수 있는 보고서를 생성할 수 있다.
